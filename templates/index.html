Panduan Deployment

Langkah-langkah untuk mendeploy aplikasi Flask ke Railway (PaaS gratis)

1

Persiapan Aplikasi

Pastikan aplikasi Flask Anda sudah memiliki file requirements.txt, Procfile, dan sudah diinisialisasi dengan Git. File requirements.txt harus berisi semua dependensi yang diperlukan.

2

Buat Akun Railway

Daftar akun gratis di Railway.app menggunakan GitHub, GitLab, atau email. Railway memberikan $5 credit gratis setiap bulan yang cukup untuk hosting aplikasi sederhana.

3

Deploy dari GitHub

Di dashboard Railway, pilih "New Project" â†’ "Deploy from GitHub repo". Pilih repository Anda dan Railway akan otomatis mendeteksi aplikasi Flask dan melakukan deployment.

4

Konfigurasi Environment

Atur environment variables jika diperlukan melalui dashboard Railway. Untuk Flask, pastikan FLASK_ENV=production dan Railway akan otomatis menangani PORT binding.

5

Akses Aplikasi

Setelah deployment selesai, Railway akan memberikan URL unik untuk mengakses aplikasi Anda. URL tersebut dapat diakses secara publik dari mana saja.
